# Wierd Substance Farming

## Prerequisites

- 32x32 World Unlocked
- 32 Drones Unlocked
- A small amount of wierd substance

## Implementation

Very simple codes, it spins up a drone for each column in the world, which uses wierd substance on the grass and harvests it. If you don't have any wierd substance, use fertiliser instead, until you get a decent amount of wierd substance, and then switch back.
